# 优化检索模块：语义截断与背景语义缺失

## 简介

本章探索不同的文档分块策略以优化 RAG 系统的检索质量。包括基于中文文档章节结构的分块方法，以及在分块中添加元信息（文章标题、章节层级等）以补充背景语义的方法。通过保留文档的结构化信息，提升检索结果的准确性和上下文完整性。
