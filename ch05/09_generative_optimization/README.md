# 优化生成模块

## 简介

本章介绍生成模块的提示工程优化技术。通过优化提示词设计（包含思维链步骤分解、Few-shot 示例、结构化输出等），引导大语言模型从检索到的上下文中筛选有用信息并生成更高质量的回答，降低幻觉率，提升生成结果的忠实度。
