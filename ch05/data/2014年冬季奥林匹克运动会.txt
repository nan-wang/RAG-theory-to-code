第二十二届冬季奥林匹克运动会（英语：the XXII Olympic Winter Games，法语：les XXIIes Jeux olympiques d'hiver，俄语：XXII Олимпийские зимние игры），一般称为索契冬奥会，于2014年2月7日至2月23日在俄罗斯索契举行，是俄罗斯继1980年夏季奥运后第二次举办奥运会、以及首次举办的冬季奥林匹克运动会。
这届赛事存在了不少问题，尤其主办国俄罗斯有被揭露在国家支持下选手使用禁药，以及会期间发动军事干涉克里米亚的争议，也使该届蒙上阴影。


== 申办过程 ==

申办初期，国际奥林匹克委员会共收到了来自七个城市的主办申请。2006年6月22日，国际奥委会执行委员会举行会议，确定了三个城市的最终候选资格，它们是俄罗斯索契、奥地利萨尔茨堡和韩国平昌。

2007年7月4日，在国际奥委会第119次全体会议上，在瓜地马拉市当地时间下午3时36分开始进行投票。在首轮的投票中，奥地利萨尔茨堡落选；当地下午3时45分，第二轮的投票结束。当地时间下午5时国际奥委会主席罗格宣布俄罗斯索契成功申办2014年冬季奥林匹克运动会的主办权。


== 圣火传递 ==

圣火传递由2013年10月7日启动，至开幕日2014年2月7日结束，共持续123天，从莫斯科到索契，横跨俄罗斯83个联邦的2900个城镇和村庄，传递方式包括步行、汽车、火车、飞机和乘马车，总里程超过6.5万公里，是冬奥会史上路程最长的火炬传递。圣火采集仪式于2013年9月29日在奥林匹亚举行，希腊高山滑雪运动员安东尼乌担任首棒火炬手，俄罗斯冰球明星亚历山大·奥韦奇金担任负责接棒。
2013年10月20日，在核动力破冰船的帮助下，圣火第一次抵达北极。
2013年11月6日，国际空间站的宇航员乘坐“联盟TMA-09M”载人飞船将圣火带往太空，11日返回地球。
圣火还到达欧洲最高峰厄尔布鲁士山和西伯利亚最深的地方贝加尔湖。

== 参赛国家与地区 ==
有88个国家与地区至少有一名运动员取得参赛资格。多米尼克、马尔他、巴拉圭、东帝汶、多哥、东加和辛巴威等七个国家都将首次在冬奥亮相。另一方面，尽管南非、波多黎各和阿尔及利亚均有一名运动员取得参赛资格，但这些国家的奥林匹克委员会表示不会参加本届冬季奥运会。

印度的运动员于将使用奥会会旗参赛，因为2012年12月印度奥委会因为选举的问题被暂停会籍。后来印度奥委会被恢复，并被允许参赛。


== 开幕及闭幕礼 ==


=== 开幕礼 ===

本届冬季奥林匹克运动会开幕礼已于当地时间2014年2月7日于菲什特奥林匹克体育场举行。来自艺术团体3000多人在开幕式上参加了舞蹈，芭蕾，杂技和马戏的表演，体育场内有40,000观众在观看开幕礼。
花样滑冰运动员伊琳娜·罗德尼娜和冰上曲棍球选手弗拉季·特列季亚克点燃火炬。


=== 闭幕礼 ===
闭幕式于索契时间2014年2月23日20:14到22:25之间在菲什特奥林匹克体育场举行，以欧洲大陆的视角展现俄罗斯文化，尤里·巴什梅特、瓦列里·杰基耶夫、丹尼斯·马祖耶夫、格兹玛娃、塔蒂亚娜·姗姆等众多俄罗斯明星现场表演。
闭幕式开始时有意模仿开幕式时的意外：雪花慢慢张开组成奥运五环，但是其中一朵雪花没有打开的情形。但是表演的最后一刻，那一环被打开。约22点10分，三个奥运吉祥物进场。随后圣火熄灭。闭幕式上，五环旗从移交到2018年冬季奥林匹克运动会举办城市平昌。


== 比赛项目 ==

本届冬奥赛事共有7个运动15大项98小项。三个滑冰项目包括花式滑冰、竞速滑冰和短道竞速滑冰。六个滑雪项目包括高山滑雪、越野滑雪、自由式滑雪、北欧混合式滑雪、跳台滑雪和单板滑雪。两个雪车项目雪车和钢架雪车。其他四个项目是冬季两项、冰壶、冰球和雪橇。总共有12个新项目登场，使本届赛会是历年来最大。

2011年4月6日，国际奥会接受了一些国际运动总会提交的项目，被考虑纳入奥运正式项目。这些项目包括：

花式滑冰团体项目
女子跳台滑雪
混合接力冬季两项
半管滑雪
团体接力雪橇
2006年11月28日，国际奥会执行委员会宣布决定不会将下列体育项目列入审查。

登山滑雪
滑雪定向
冬季铁人三项
