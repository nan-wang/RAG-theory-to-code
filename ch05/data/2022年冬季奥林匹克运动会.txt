第二十四届冬季奥林匹克运动会（英语：the XXIV Olympic Winter Games；法语：les XXIVes Jeux olympiques d'hiver），即2022年冬季奥林匹克运动会（2022 Winter Olympics），通称北京2022，是于2022年2月4日至2月20日在中国北京及周边地区举行的国际冬季综合运动会，部分项目的比赛从2022年2月2日开始。河北省西北部城市张家口也承办了本届冬奥会大部分户外冰雪项目。这是中国继2008年夏季奥林匹克运动会后第二次举办奥运会，亦是中国首次举办冬季奥运会，而作为主办地点之一的北京也因而成为首座既举办过夏季奥运会，又举办过冬季奥运会的“双奥之城”。本次冬奥会与2018年平昌冬奥会、2020年东京夏奥会连续在亚洲举行，是奥运会历史上首次在欧洲以外的大洲连续举行奥运会。
由于2020年东京夏季奥林匹克运动会延期一年举行，与北京冬奥会的间隔缩短为半年，这是1994年冬季奥运会与夏季奥运会错开举办以来，间隔最短的一次。
国际奥林匹克委员会执行委员会于2014年7月确定了六个候选城市，但其中的四个城市因各种原因纷纷退出，最后只有中国北京和哈萨克斯坦阿拉木图相互竞争。2015年7月31日在马来西亚首都吉隆坡举行的第128届国际奥委会全体会议上宣布由北京主办。


== 申办 ==

2022年冬季奥林匹克运动会的提交申办截止时间为2013年11月14日。国际奥林匹克委员会于11月15日公布了六个提出申办的城市名单，后因瑞典斯德哥尔摩于2014年1月17日宣布退出申办，波兰克拉科夫于2014年5月退出申办，乌克兰利沃夫于2014年6月退出申办，使申办城市减少为三个。
2014年7月7日，国际奥委会在洛桑宣布了三个候选城市：挪威奥斯陆、哈萨克斯坦阿拉木图和中国北京。根据计划，国际奥委会将选出三个候选城市。但由于只剩下三个申办城市，因此没有城市被淘汰。
2014年10月1日，奥斯陆宣布放弃2022年冬奥会的申办，使申办城市减少为2个。
2015年7月31日在马来西亚首都吉隆坡举行的第128届国际奥委会全体会议上决定由北京主办。


== 宣传 ==


=== 会徽 ===

本届冬奥会的会徽于2017年12月15日公布。会徽以“冬梦”为主题，以汉字“冬”为灵感来源，主色调为蓝色，同时使用源自中华人民共和国国旗的红黄两色。


=== 吉祥物 ===

该届冬奥的吉祥物“冰墩墩”于2019年9月17日公布。“冰墩墩”以中国国宝熊猫为原型进行设计，结合熊猫形象与冰晶外壳，体现冰雪运动和现代科技，整体形象类似航天员，寓意创造非凡、探索未来，体现了追求卓越、引领时代，以及面向未来的无限可能。


=== 口号 ===
2021年9月17日，北京冬奥及冬残奥组委会在首都博物馆宣布该届比赛的口号为“一起向未来”（英语：Together for a Shared Future）。根据官方解释，口号的含义是“在奥林匹克精神的感召下，（中国人民）与世界人民携手共进、守望相助、共创美好未来”。该口号中“未来”的官方英译并非常见的“(the) Future”，而是“a Shared Future”（直译为“共享的未来”），与中共中央总书记习近平提出的“人类命运共同体”（英语：Community with a Shared Future for Mankind）理念中“命运”的官方英译“a Shared Future”相同。中国官方称，这体现了“人类命运共同体的理念同奥林匹克价值观的契合，是人类命运共同体理论对奥林匹克价值观的丰富发展”。


=== 奖牌 ===

2022年冬奥的奖牌设计于2021年10月26日在北京2022年冬奥会倒计时100天主题活动仪式上发布。本届冬奥会奖牌由圆环加圆心构成牌体，形象来源于中国古代同心圆玉璧，共设五环，承袭了2008年夏季奥林匹克运动会的奖牌设计。奖牌正面中心刻有奥林匹克标志，周围刻有该届冬奥英文全称“XXIV Olympic Winter Games Beijing 2022”字样。打凹处理的圆环取意传统弦纹玉璧，装饰纹样来自中国传统纹样。奖牌背面中心刻有该届冬奥会会徽，周围刻有该届冬奥中文正式名称“北京2022年第24届冬季奥林匹克运动会”字样。圆环上的24个点及运动弧线取意古代天文图，象征着浩瀚星空、人与自然的和谐，及如群星璀璨的参赛运动员。奖牌背面最外环镌刻获奖运动员对应的比赛小项名称。


=== 体育项目图标 ===
该届冬奥的共24个体育图标于2020年12月31日晚的“2021迎冬奥环球跨年冰雪盛典”上正式发布。图标设计的灵感来源为中国汉字，主要呈现形式为篆刻艺术，结合冬季运动元素与中国传统文化。图标的霞光红底色寓意日出东方及比赛在农历春节期间举办的喜庆气氛。

== 火炬及火炬接力 ==

该届冬奥的火炬“飞扬”于2021年2月4日晚，在国家游泳中心举行的倒计时一周年活动上公布。该届冬奥火炬以氢气为燃料，颜色为红色和银色，并附有祥云纹样和雪花图案，红色线条象征冰雪赛道及永恒的火种。此次采用的火炬为中国航天科技集团第六研究院研制的高压储氢火炬，燃烧器出口处安装有喷涂碱金属的格栅，以提供焰色反应。
圣火采集仪式于2021年10月18日在希腊古代奥林匹克运动会遗址举行，曾参加2018年平昌冬奥会的希腊高山滑雪运动员雅尼斯·安东尼奥担任本届冬奥会的首位火炬手。次日在帕那辛纳克体育场举办圣火交接仪式，国际奥委会副主席于再清从希腊奥委会主席斯皮洛斯·卡普拉洛斯手中接过圣火后将其送往中国，10月20日凌晨抵达北京首都国际机场，同日在北京奥林匹克塔举行火种欢迎仪式。
此次火炬接力分为火种展示和传统火炬传递两部分，其中传统火炬传递受2019冠状病毒病疫情影响仅在该届冬奥的三个赛区进行，时间安排在2022年2月2日至4日。


== 开闭幕式 ==

该届冬奥的开闭幕式于国家体育场（鸟巢）举办。开幕式于北京当地时间2月4日举行，举办日期恰逢中国农历新年正月初四及传统节气立春，开幕式节目以“构建人类命运共同体”为核心表达，以“简约、安全、精彩”为创作原则，加上2019冠状病毒病疫情防疫需要、此次冬奥会开幕式时长、人员数量相较往届奥运会均有一定程度的缩减。闭幕式则于北京当地时间2月20日举行。

=== 防疫安排 ===

2021年9月29日，国际奥委会在执行委员会会议中审议了2022年冬奥及冬残奥疫情防控关键政策，包括：相关场所自2022年1月23日至2022年冬残奥会结束期间实施闭环管理；未获医学豁免的相关人员如果入境中国大陆14天前已完成疫苗接种，则无需接受隔离，否则入境后须接受21天强制隔离；仅面向居住在中国大陆境内符合疫情防控相关要求的观众售票；除运动员和部分随队官员入住冬奥村外，其他所有注册人员都将入住该届冬奥组委会签约饭店；所有进入闭环内的境内外参与人员和工作人员必须每天接受一次病毒检测。2022年1月17日，北京冬奥组委会表示，因疫情防控形势严峻复杂，决定不公开销售门票，改为定向组织观众现场观看比赛，现场观赛观众须严格遵循疫情防控要求。

交通运输方面，冬奥相关人员只能乘坐专用车辆及专用高铁列车来往冬奥各个场馆。其中专用车辆贴有专用标识以供大众辨认，北京市交通管理部门要求非相关人员注意与冬奥专用车辆及车内人员保持安全距离，如发生事故须等待专业人员到场处置。京张高铁的沿线车站和冬奥专用列车亦在冬奥期间设立专供冬奥相关人员活动的区域，确保冬奥相关人员与外界间没有任何接触。另外，承接大量冬奥相关人员出入境的北京首都机场亦划设专用通道，确保普通旅客不会与出入境相关人员接触。航班方面，北京当局于2021年12月确定以包机和临时航班为主，商业航班为辅的原则，来确保所有境外相关人员的入境，出境则仅能乘坐包机或临时航班。奥林匹克专用车道于2022年1月21日正式启用。
2021年10月25日，北京冬奥组委发布第一版防疫手册，第二版防疫手册于同年12月13日发布。
据NOW新闻台报道，河北省因应疫情防控工作指已制定专项工作方案，就医疗、采样检测等组建八支防疫队伍，并将筹建方舱实验室，预计赛时核酸检测量每日将达四万，比赛期间严格实施闭环管理，不遵守防疫规定的相关人员可能会被取消进入闭环管理场所的资格。
在奥运会场所的每个人（包括运动员、工作人员、观众）都被要求安装冬奥通（My 2022）手机应用程序。My 2022用来出示健康码、提交每日健康申报、记录行程等。该APP也提供赛事新闻和有聊天功能，但有媒体指责该app存在隐私泄露问题和安全漏洞，部分媒体称其为间谍app。。国际奥委会回应称，应用访问麦克风，日历等隐私权限前需要经过用户授权，用户拥有最高的权限。该应用也通过了Google Play和苹果App Store的审核规则。

== 参赛代表团 ==
世界反兴奋剂组织于2019年12月9日宣布所有俄罗斯运动员仅能以个人名义参加国际主要比赛。俄罗斯方面不满这一处罚，决定上诉至国际体育仲裁法庭。最终国际体育仲裁法庭于2020年12月17日宣布俄罗斯方面的上诉无效，但将世界反兴奋剂组织对俄罗斯的禁令由四年缩短为两年。禁令生效期间，俄罗斯运动员不得以国家名义参与奥林匹克运动会、残疾人奥林匹克运动会和世界锦标赛等国际大型赛事。此次比赛，俄罗斯运动员以“俄罗斯奥林匹克委员会”（Russian Olympic Committee）之英文缩写“ROC”的名义参赛，若该国选手获得金牌，颁奖仪式会播放彼得·柴可夫斯基《第一钢琴协奏曲》第一乐章选段。
2021年9月8日，国际奥林匹克委员会通过对朝鲜奥林匹克委员会的纪律处分，因该国奥委会以2019冠状病毒病疫情状况为由单方面缺席2020年夏季奥林匹克运动会，违反了《奥林匹克宪章》，故暂停其活动并禁止财政支持至2022年底；因此，朝鲜被禁止以国家名义参加本届奥运会，运动员可以以个人名义在奥林匹克旗下参赛。2022年1月5日，朝鲜方面致信中国奥委会称“因为敌对势力的阴谋活动和全球传染病大流行，朝鲜虽不能参加本届奥运会，但朝鲜将全方位支持和声援中国要举办一届隆重精彩的奥运盛会的一切事宜”。
共有91个国家和地区（含俄罗斯奥委会代表队）的代表团参加该届冬奥，其中海地和沙特阿拉伯是首次亮相冬季奥运会。肯尼亚原本有一名选手取得资格，但该选手后来因为疫情原因放弃参赛。

== 转播单位 ==

2022年冬奥会的转播权在一些国家已作为长期转播权交易售出。在法国和英国，2022年冬奥会将是探索传播首届也是唯一一届持有泛欧版权的赛事（因为在本届奥运会举办一个月后后探索传播与华纳媒体合并为华纳兄弟探索，奥运转播权也自动转移到了该新公司下）。在英国，为遵守体育赛事转播的相关法规，探索传播需要再授权地面网络转播，如前版权持有人英国广播公司。在中国大陆与澳门地区，中国中央广播电视总台为该届冬奥的独家转播商，并基于版权分授协议授权中国移动咪咕、腾讯和快手等享有在中国大陆地区传播2022年北京冬奥会内容的权利。


== 转播数据 ==
2022年10月20日，国际奥委会发布报告，表示共有创纪录的20.1亿人透过广播电视和数位平台观看了2022年北京冬奥会，比4年前的平昌冬奥会观看人数增长5％。报告亦表示，本届赛事成为数位化参与程度最高的一届冬奥会。而受访的参赛运动员对北京冬奥会也大加赞赏，对赛事采取的防疫措施感到满意。


== 国际奥委会授奖 ==
2022年2月19日，即本届冬奥会闭幕前一天，在国际奥委会第139次全会上，国际奥委会主席托马斯·巴赫宣布，为感谢中国人民对于北京冬奥会做出的卓越贡献，使北京冬奥会获得卓越的成功，将奥林匹克杯授予全体中国人民。


== 争议 ==

肆虐全球的新冠肺炎疫情对赛事造成了相当的影响，部分项目预选赛取消；原本面向中国大陆低风险人群公开售票的计划也取消，改为邀请特定群体观赛。
另外，政治纷争、防疫制度、比赛的判罚尺度、奥运村的接待工作、休战协议等也成为争论焦点。
