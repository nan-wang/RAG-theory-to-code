## Chinese terminology glossary (for consistency)

- Runnable = Runnable（可运行对象）
- invoke/ainvoke = 同步/异步调用
- stream/astream = 同步/异步流式输出
- batch/abatch = 同步/异步批量调用
- State = 状态, Node = 节点, Edge = 边
- Conditional Edge = 条件边
- Checkpoint = 检查点
- Reducer = 归约器
- Document = 文档, Embedding = 嵌入, VectorStore = 向量存储
